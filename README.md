# 🚀 Ken Redel Rante – Professional Portfolio

This project is a React portfolio built with Vite for Ken Redel Rante. It features Ken's auditing and IT consulting expertise, projects, resume download, and a working contact form.

---

## 🚀 What's included

- Vite + React app structure
- Responsive mobile navigation with hamburger menu
- Smooth scrolling and active section states
- Interactive project cards and animated cursor effects
- Netlify-compatible contact form
- Direct resume download from public assets
- Modern responsive layout with improved metadata

---

## 🧩 Installation

```bash
npm install
npm run dev
```

---

## 📄 Resume

The resume file is now served from `public/Resume_Ken_Rante.pdf`.

Use the download button on the homepage or open the PDF directly.

---

## 📬 Contact form

The contact form is configured for Netlify forms by default.
If you deploy to Netlify, the form will work automatically.

For a non-Netlify fallback, use the optional Formspree endpoint in `.env`.

---

## 🛠️ Key files

- `index.html` — Vite HTML entry
- `src/App.jsx` — main React app
- `src/main.jsx` — React entry point
- `src/styles.css` — site styling
- `package.json` — dependencies and scripts
- `public/` — static assets and resume PDF
